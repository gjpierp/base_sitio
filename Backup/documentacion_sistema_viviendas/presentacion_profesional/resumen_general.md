# Resumen General del Sistema de Viviendas Fiscales FACH

Este sistema gestiona la administración, asignación, arriendo y control de viviendas fiscales, pabellones de soltero, servicios básicos, seguros, integración con sistemas externos (SIAP, SISREM, empresas de servicios, UF), gestión documental, usuarios, roles, permisos y puntuación de Oficiales y PCP.

## Funcionalidades principales

- Registro y asignación de viviendas fiscales y pabellones
- Gestión de beneficiarios, familiares y ocupantes
- Control de servicios básicos y facturación automática
- Gestión de seguros en UF y CLP para bienes muebles e inmuebles
- Integración con SIAP, SISREM, empresas de servicios y sistema de UF
- Generación y almacenamiento automático de documentos y contratos
- Módulo de usuarios, roles, permisos y bitácora
- Parametrización y mantenimiento de catálogos
- Registro y cálculo de puntuación para Oficiales y PCP
- Auditoría, reportes y trazabilidad completa

## Arquitectura

- Base de datos relacional (MySQL, Oracle, PostgreSQL) y soporte para MongoDB
- API RESTful para integración y automatización
- Seguridad basada en roles y permisos
- Documentación profesional, diagramas y scripts listos para implementación
